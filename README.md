# 🚴‍♂️ Bikcraft

Projeto desenvolvido como parte dos estudos de **HTML e CSS** no curso da Origamid.  
O objetivo foi criar um site fictício para uma empresa de bicicletas personalizadas, aplicando conceitos de estruturação semântica, estilização e boas práticas de desenvolvimento front-end.

---

## 📑 Conteúdo do Projeto

- **index.html** → Página inicial com apresentação da marca  
- **bicicletas.html** → Catálogo de modelos de bicicletas  
- **seguros.html** → Informações sobre planos de seguro  
- **orcamento.html** → Formulário para solicitação de orçamento  
- **contato.html** → Página de contato  
- **termos.html** → Termos e condições  

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** → Estrutura semântica das páginas  
- **CSS3** → Estilização e responsividade  
- **JavaScript** → Interatividade básica  
- **Favicon e imagens otimizadas**  

---

## 🎯 Aprendizados

- Organização de código e semântica em HTML  
- Uso de classes e seletores avançados em CSS  
- Criação de layouts responsivos  
- Estruturação de um projeto completo com múltiplas páginas
